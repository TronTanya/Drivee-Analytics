# Презентация и макеты (Drivee Analytics)

Материалы для **защиты**, **жюри** и **онбординга** команды: слайды в PowerPoint и ссылка на макеты в Figma.

---

## Слайды (PowerPoint)

| Файл | Действие на GitHub |
|------|-------------------|
| **[`prezentaciya.pptx`](./prezentaciya.pptx)** (~12.6 МБ) | Откройте файл в репозитории → кнопка **Download raw file** (или **⋯** → *Download*) |

Локально после клонирования:

```bash
open docs/presentations/prezentaciya.pptx   # macOS
```

Исходник на рабочем столе при необходимости обновляйте вручную и снова копируйте в `docs/presentations/prezentaciya.pptx`, затем коммит.

## Как это выглядит на GitHub

1. В дереве файлов: **`docs` → `presentations` → `README.md`** — эта страница рендерится как документация.
2. Рядом лежит **`prezentaciya.pptx`** — бинарный файл; предпросмотр слайдов GitHub не показывает, только скачивание.
3. В корневом **[`README.md`](../../README.md)** есть блок «Презентация и дизайн» с быстрыми ссылками.
